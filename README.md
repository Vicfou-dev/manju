## 📖 About

Reading manga in another language should be effortless.

**Manju** automatically detects speech bubbles, extracts the original text, translates it and redraws the translated text while preserving the artwork.

Instead of simply overlaying translated text, Manju recreates a natural reading experience using AI-based image restoration and rendering.

## 🧠 How it works

The translation pipeline is composed of several AI steps:

```
Image
   │
   ▼
Speech Bubble Detection
   │
   ▼
OCR
   │
   ▼
AI Translation
   │
   ▼
Text Removal (Inpainting)
   │
   ▼
Automatic Text Rendering
   │
   ▼
Translated Image
```

## 💡 Why Manju?

Unlike traditional OCR translators, Manju aims to recreate a natural reading experience.

Instead of simply replacing text, the application:

- preserves the artwork
- respects speech bubbles
- keeps natural typography
- minimizes visual artifacts
- provides high-quality AI translation
